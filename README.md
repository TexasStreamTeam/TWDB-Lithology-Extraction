# TWDB-Lithology-Extraction
Streamlines the input of lithology descriptions from the TWDB report viewer. For Krause Springs Phase III 600+ SDR wells did not have lithology descriptions within the downloadable database. This script took the URL from the well data spreadsheet, accesses the report on the website, and extracts the lithology description into a spreadsheet in the appropriate format. 
Saves a ton of time and limits tedious work. Can be used to replicate similar data extraction processes for future projects where well data is extracted from TWDB well reports.
